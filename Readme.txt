Titlu: Android - Think Different
Proiect realizat de Dorobantu Marius Alexandru, elev al C.N. Carol I.
Clasa: XI-C
Profesor indrumator: Monica Cerbulescu

Libraries utilizate: MaterializeCSS
Realizat in HTML5, CSS, Javascript si jQuery.

Am creat acest site din pasiunea mea pentru sistemul de operare Android, dezvoltat de Google si din dorinta de a informa oamenii. 
Site-ul contine multe informatii despre programul Android One, telefoanele oficiale Google, un istoric al versiunilor
sistemului de operare si o scurta demonstratie a stilului de design Material Design, stil pe care, de altfel, il si urmeaza intregul site. 

Parte din CSS si Javascript este si MaterializeCSS, un proiect open-source sustinut de Google pentru efecte Material Design.
